# Business Consulting Website

A modern, responsive static website for a business consulting firm, built using HTML and CSS. This project showcases services, projects, client testimonials, and company information in a clean and professional layout.

## Features

- **Responsive Design**: Optimized for desktop and mobile devices.
- **Navigation Bar**: Easy-to-use menu with links to different sections.
- **Hero Section**: Eye-catching banner with call-to-action button.
- **Services Overview**: Highlights key services like consulting, projects, and client management.
- **About Us**: Detailed section about the company's mission and story.
- **Our Services**: Visual gallery of service offerings with images.
- **Client Testimonials**: Logos and SVGs representing satisfied clients.
- **Custom Styling**: Uses Google Fonts (Poppins and Montserrat) for a professional look.

## Technologies Used

- **HTML5**: Structure and content of the website.
- **CSS3**: Styling, layout, and responsiveness.
- **Google Fonts**: Poppins and Montserrat for typography.

## Installation and Setup

Since this is a static website, no server setup is required. Simply clone the repository and open the `index.html` file in your web browser.

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/business-website-main.git
   ```

2. Navigate to the project directory:
   ```
   cd business-website-main
   ```

3. Open `index.html` in your preferred web browser.

## Usage

- Browse the website sections using the navigation bar.
- Click on "Free Consultation" or "More Info" buttons for interactive elements (currently placeholders).
- View the responsive design by resizing your browser window or viewing on mobile devices.
